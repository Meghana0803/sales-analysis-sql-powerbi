# Sales Performance Analysis using SQL & Power BI

## Overview
Analyzed sales data to identify revenue trends, top-performing products, and regional performance using SQL and Power BI.

## Tools Used
- Microsoft SQL Server
- Excel
- Power BI Desktop

## Key Analysis
- Monthly and yearly sales trends
- Top products by revenue
- Region-wise performance
- KPI metrics (Total Sales, Profit, Growth)

## Power BI Dashboard
The interactive Power BI dashboard is included as a `.pbix` file.

### How to View
1. Download Power BI Desktop
2. Open the `.pbix` file from this repository
3. Use filters and slicers to explore insights

## SQL Skills Demonstrated
- Joins
- Group By & Aggregations
- Date-based analysis
- Business KPIs

## Outcome
Built an end-to-end sales analysis solution to support data-driven decisions.
